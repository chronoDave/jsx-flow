## Getting Started

### Installation

```sh
npm i jsx-flow
```

### Example

Using **jsx-flow** should feel familiar for those used to writing JSX:

```TSX
const Template = props => (
  <html lang="en">
    <body>{props.children}</body>
  </html>
);

const Page = () => (
  <Template>
    <h1>Title</h1>
  </Template>
)
```
